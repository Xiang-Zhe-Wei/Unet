# U-Net Semantic Segmentation (PyTorch)

This repository implements **U-Net** for semantic segmentation using the VOC2012 dataset, with a custom PyTorch `Dataset` for loading images and segmentation masks.

---

## Features
- Custom dataset loader for VOC2012
- Loads both **JPEGImages** (input) and **SegmentationClass** (mask)
- Designed for U-Net training
- Easy to extend or modify
